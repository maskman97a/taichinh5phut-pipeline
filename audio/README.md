# Background Music Folder

Đặt các file `.mp3` nhạc nền vào đây. Pipeline sẽ random chọn 1 file mỗi video.

## Nơi tải nhạc free royalty-free:

1. **Pixabay Music** (recommend): https://pixabay.com/music/
   - Free, không cần signup
   - Tải MP3 trực tiếp
   - Filter: Genre = Cinematic/Lofi/Corporate
   - Đề xuất search: "corporate", "motivational", "lofi study", "inspiring"

2. **YouTube Audio Library**: https://www.youtube.com/audiolibrary
   - Cần login YouTube
   - 100% safe cho monetize

3. **Free Music Archive**: https://freemusicarchive.org/

## Khuyến nghị

Tải 5-10 track, mỗi track 2-3 phút. Pipeline sẽ chọn random.

## Yêu cầu

- Format: MP3 (khuyến nghị 128-192 kbps)
- Length: ≥ 30 giây
- Volume sẽ tự động giảm xuống 15% khi mix với voice

## Đặt tên file (tuỳ ý)

bgm_02_corporate.mp3
